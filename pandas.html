Map values using a dictionary

Create a new column in a dataframe using a dictionary with keys that are entries in one of the columns. For example mapping a nation's capital to a dataframe containing the country name.

  dict[key]=value
  df[new_col] = df[col].map(dict)
		

  caps['Italy']='Rome'
  df['Capitals'] = df['Country'].map(caps)
